Rules of the game

The game can be run from the main class called Drooms. The Javadoc gives you all you need to know.

The game currently has only one implementation, which is called DefaultGame. Study the JavaDocs to find out the rules of that game.

Strategies

Your strategies must implement the Strategy interface. For acceptance testing of your strategies, you would be well-advised to use DroomsTestHelper class. DecisionMaker class will help you understand what you can expect inside your strategies.

Using non-stable APIs in your strategies is at your own risk. The following APIs are considered to be stable:

  • Everything in the API package.
  • Everything in the Event package.
  • Everything in the Fact package.
