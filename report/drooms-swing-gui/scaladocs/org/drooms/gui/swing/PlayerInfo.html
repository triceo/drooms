PlayerInfo

case class PlayerInfo(name: String, jar: Option[File], strategyDir: Option[File], strategyClass: String) extends Product with Serializable

Represents an information about player that is needed for the DroomsGame.

Player info contains:

  • player's name
  • [optional] jar file with the strategy
  • [optional] directory with stragegy classes and resources
  • FQN of the main strategy class
