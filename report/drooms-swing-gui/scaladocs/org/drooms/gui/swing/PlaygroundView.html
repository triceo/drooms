org.drooms.gui.swing

PlaygroundView

class PlaygroundView extends ScrollPane with Reactor with Logging

Represents the Playground in GUI as ScrollPane.

Playground contains the grid (table) of all nodes and also some additional GUI elements like border from walls or labels for the rows/columns numbers.
